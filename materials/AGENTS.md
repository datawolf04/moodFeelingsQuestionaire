# materials

Research instruments and validation papers for the **Mood and Feelings Questionnaire (MFQ)**.

## Contents

- **MFQ forms** — child self-report long, parent report long/short, MFW (Mood and Feelings Words) child short
- **Cut-point reference** — `mfqCutPointInfo.pdf`
- **Validation papers** — Angold1995a (original), Messer1995a, Eyre2021, Thabrew2018, Schlecter2023

## Notes

- No code, build system, tests, or configs. This is a static PDF collection only.
- PDFs are not generated or derived — they are committed source assets.
- File naming convention: `AuthorYear.pdf` for papers, `mfq<FormDescription>.pdf` for instruments.
